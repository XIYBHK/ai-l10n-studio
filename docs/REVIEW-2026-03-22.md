**当前代码**:

```rust

**修复**: 将全局注册表的锁类型从 `std::sync::RwLock` 改为 `parking_lot::RwLock`（项目已有此依赖）。parking_lot 的锁不会中毒（poisoned），无需 `.expect()`：

let guard = registry.read();
let mut guard = registry.write();
```

**注意**: 同时需修改 `OnceLock` 初始化处的类型声明。

### Task 8: config_draft.rs - apply_secrets 的 zip 长度不匹配

**文件**: `src-tauri/src/services/config_draft.rs:414-424`

**当前代码**:

```rust
fn apply_secrets(config: &mut AppConfig, secrets: &ConfigSecrets) {
    if !secrets.api_key.is_empty() {
        config.api_key = secrets.api_key.clone();
    }
    // zip() 会在较短一方耗尽时停止，静默丢弃多余数据
    for (ai_config, api_key) in config.ai_configs.iter_mut().zip(secrets.ai_api_keys.iter()) {

        }
    }
}
```

**修复**: 改为按索引遍历，处理长度不一致的情况：

```rust
fn apply_secrets(config: &mut AppConfig, secrets: &ConfigSecrets) {
    if !secrets.api_key.is_empty() {
        config.api_key = secrets.api_key.clone();
    }
    for (i, ai_config) in config.ai_configs.iter_mut().enumerate() {
        if let Some(api_key) = secrets.ai_api_keys.get(i) {
            if !api_key.is_empty() {
                ai_config.api_key = api_key.clone();
            }
        }
    }
}
```
